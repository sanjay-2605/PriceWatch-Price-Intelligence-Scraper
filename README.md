# PriceWatch — Price Intelligence Scraper

## Features
- Multi-platform scraping (Amazon, Flipkart)
- Anti-bot evasion (UA rotation, delays)
- REST APIs using FastAPI
- AI anomaly detection (z-score)

## Tech Stack
Python, FastAPI, BeautifulSoup, SQLAlchemy

## Run
pip install -r requirements.txt  
uvicorn app.main:app --reload
